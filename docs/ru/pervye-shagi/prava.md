# Права бота

FistashkinBot требует определённых прав для корректной работы.

{% hint style="warning" %}
Без нужных прав часть команд будет недоступна или вызовет ошибку.
{% endhint %}

{% hint style="info" %}
Вы можете разрешить использовать определенные команды только в нужных каналах или для определенных пользователей.
{% endhint %}

**1.** Для этого следует перейти в настройки сервера с **компьютерной** версии Discord в раздел **Интеграция.**
**2.** Следующим шагом следует найти **FistashkinBot** в списках ботов и нажать **Управление**, чтобы перейти в режим редактирования.
**3.** После этого перед вами откроется окно со **всеми командами**. Вы можете редактировать различные условия для использования команд и прочее.


## Обязательные права

| Право | Описание |
|---|---|
| Просмотр каналов | Бот должен видеть каналы |
| Отправка сообщений | Ответы на команды |
| Встроить ссылки | Embed-сообщения |
| Прикреплять файлы | Изображения и файлы |
| Читать историю сообщений | Некоторые команды |
| Добавлять реакции | Интерактивные меню |

## Права для модерации

| Право | Команда |
|---|---|
| Кикать участников | `/kick` |
| Банить участников | `/ban` |
| Мьютить участников | `/mute` |
| Выдавать предупреждения | `/warn` (3 из 3 варнов = бан) |
| Управлять сообщениями | `/purge` |
| Управлять ролями | `/role` |
| Управлять каналами | `/delay` |
