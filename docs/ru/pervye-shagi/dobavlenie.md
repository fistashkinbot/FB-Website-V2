# Добавление бота

Это руководство поможет тебе быстро добавить FistashkinBot на свой сервер.

{% hint style="warning" %}
Для добавления бота необходимо право **Управление сервером** на целевом сервере.
{% endhint %}

## Шаги

1. Перейди по [ссылке для приглашения](https://fistashkinbot.xyz/invite)
2. Выбери сервер из выпадающего списка
3. Нажми **Авторизовать** и пройди капчу
4. Готово! Бот появится на сервере 🎉

## После добавления

Убедись, что у бота есть нужные права. Начни работу с команды:

```
/help
```
