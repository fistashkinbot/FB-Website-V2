# Хранимые данные

FistashkinBot хранит только необходимые данные для работы функционала.

{% hint style="info" %}
Мы не продаём и не передаём ваши данные третьим лицам. Для правильной работы бота мы храним определенные данные.
{% endhint %}

## Что хранится?

| Данные | Зачем |
|---|---|
| ID пользователя Discord | Функционирование Бота / Модерация / Экономика
| ID сервера | Привязка настроек бота к серверу / **AutoMod** |
| ID каналов | Временные голосовые каналы / Журнал Аудита |
| ID ролей | Магазин ролей / Журнал аудита |
| ID сообщений | Журнал Аудита ( Сообщение было **отредактировано**/**удалено** ) |
| Данные команд Discord API | Обработка запросов |

## Что НЕ хранится?

- Содержимое сообщений
- Личная информация (Email, пароли, файлы Cookies)
- История голосовых сессий

## Удаление данных

Для удаления всех данных сервера напиши в поддержку: [discord.gg/H9XCZSReMj](https://discord.gg/H9XCZSReMj)
Данные будут удалены в течение **3 дней**

{% hint style="danger" %}
Удаление данных необратимо и восстановлению не подлежит.
{% endhint %}
