## Версия 1.8
{% hint style="warning" %}
Некоторые изменения версии 1.8 уже реализованы в боте.
{% endhint %}

### Добавлено / Новые команды
- `угадай` — Угадывание объектов по картинке (игры, города, логотипы, страны, транспорт)
- `настройка автомод` — Управление правилами автомодерации бота
- `мониторинг` — Список мониторингов, на которых присутствует бот для продвижения
- `плей` — Воспроизведение музыки в голосовом канале
- `стоп` — Остановка музыкального плеера
- `громкость` — Установка громкости плеера
- `рольинфо` — Отображение информации о роли (цвет, позиция и другие параметры)
- `эмоция` — Показ случайного эмодзи с серверов, на которых присутствует бот
- `реакция` — Эмоциональные реакции (заплакать, покраснеть, потанцевать и другие)

### Изменено
- Команда `рп` переименована в `взаимодействие`
- Команда `слот` переработана: теперь перед ставкой требуется подтверждение через кнопку
- В команду `стат` добавлена дополнительная информация
- В систему логирования добавлено событие **«Посещение голосового канала»**
- Логи теперь отправляются через вебхуки (при отсутствии вебхука он создаётся автоматически)
- Добавлены проверки корректности взаимодействий с кнопками
- Функция тематического оформления бота (Хеллоуин / Новый год) полностью переписана
- Значительная часть кода функций и обработчиков событий переписана
- Запросы `aiohttp` вынесены в отдельную директорию `services`
- В команду `юзер` добавлена кнопка **«Открыть Spotify»** (отображается, если участник слушает музыку)
- Команда `взаимодействие` получила ряд доработок и улучшений
- В команде `разбан` теперь отображаются только те участники, которые реально находятся в списке банов сервера
- Обработчик команд модерации полностью переписан
- Модуль `moderation.py` оптимизирован и уменьшен в размере

### API и интеграции
- **[Отменено]** Добавление wrapper'а [**FiftyAPI**](https://docs.fifty.su/) для игры `угадай`
- **[API]** Добавлены JSON-файлы из закрытой версии [**FiftyAPI**](https://docs.fifty.su/) для игры `угадай`

### Веб-интерфейс
- **[Веб]** Полностью переписан футер сайта
- **[Веб]** Слегка обновлена верхняя панель навигации
- **[Веб]** Изменён домен сайта: теперь используется [**fistashkinbot.xyz**](https://fistashkinbot.xyz) (старый домен больше недоступен)

### Музыка
- **[Музыка]** Плеер полностью переписан и доведён до рабочего состояния
- **[Музыка]** Треки теперь получаются из **SoundCloud** (вместо YouTube)
- **[Музыка]** При поиске музыки теперь доступен выбор конкретного трека из результатов
- **[Музыка]** Lavalink обновлён до версии **4.0**

### Исправлено
- Исправлены строки локализации во всех языковых файлах
- Исправлены команды `кейсы`, `бойцовский_клуб` и `монетка`
- Добавлен **Jishaku 2.6.5**

## Версия 1.7

### Добавлено / Новые команды
- `нсфв` — Просмотр NSFW аниме-контента
- `настройки логи` — Настройка канала для логирования событий
- `кубик` — Генерация случайного числа (аналог игрального кубика)
- `аниме-тян` — Просмотр картинок с аниме-тян

### Изменено
- Все команды взаимодействия объединены в одну команду `рп`
- В информации о сервере теперь отображается номер шарда (**Shard**)
- Полностью реализована система логирования на сервере со следующими событиями:
  - Присоединился новый участник
  - Участник покинул сервер
  - Никнейм участника был изменён
  - Участник получил предупреждение
  - Участник был забанен
  - Участник был изгнан (кикнут)
  - Сообщение было удалено
  - Сообщение было отредактировано
- Изменён основной цвет бота (ребрендинг)
- Изменён цвет эмодзи в команде `сервер`
- В команде `осебе` установлен лимит в **2048 символов**

### Улучшения и исправления
- **[Веб]** Множественные улучшения сайта:
  - Обновлена секция **🛠️ Мои возможности**
  - Проведён ребрендинг цвета сайта
  - Обновлено отображение эмбеда сайта в Discord
  - Обновлена библиотека FontAwesome
  - Ссылка «Добавить в Discord» теперь открывается в новой вкладке
- **[Бот]** Улучшения системы ранкинга:
  - Исправлена отправка уведомлений о повышении уровня в личные сообщения
  - Изменены математические формулы расчёта опыта
- **[Модерация]** Пользователь теперь получает уведомление в ЛС при выдаче предупреждения, таймаута, кика или бана
- Исправлена команда `юзер` — теперь корректно отображает информацию о ботах
- Удалены команды `выдать_роль`, `снять_роль` и `создать_роль`
