# Додавання бота

Цей посібник допоможе тобі швидко додати FistashkinBot на свій сервер.

{% hint style="warning" %}
Для додавання бота необхідне право **Керування сервером** на цільовому сервері.
{% endhint %}

## Кроки

1. Перейди за [посиланням для запрошення](https://fistashkinbot.xyz/invite)
2. Вибери сервер зі списку, що випадає
3. Натисни **Авторизувати** та пройди капчу
4. Готово! Бот з'явиться на сервері 🎉

## Після додавання

Переконайся, що у бота є потрібні права. Почни роботу з команди:

```
/help
```

{% hint style="success" %}
Готово! Тепер FistashkinBot у тебе на сервері. Переходь до наступних розділів для налаштування.
{% endhint %}
