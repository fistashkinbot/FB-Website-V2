# Збережені дані

FistashkinBot зберігає лише необхідні дані для роботи функціоналу.

{% hint style="info" %}
Ми не продаємо і не передаємо ваші дані третім особам. Для коректної роботи бота ми зберігаємо певні дані.
{% endhint %}

## Що зберігається?

| Дані | Навіщо |
|---|---|
| ID користувача Discord | Функціонування Бота / Модерація / Економіка |
| ID сервера | Прив'язка налаштувань бота до сервера / **AutoMod** |
| ID каналів | Тимчасові голосові канали / Журнал Аудиту |
| ID ролей | Магазин ролей / Журнал аудиту |
| ID повідомлень | Журнал Аудиту ( Повідомлення було **відредаговано**/**видалено** ) |
| Дані команд Discord API | Обробка запитів |

## Що НЕ зберігається?

- Вміст повідомлень
- Особиста інформація (Email, паролі, файли Cookies)
- Історія голосових сесій

## Видалення даних

Для видалення всіх даних сервера напишіть у підтримку: [discord.gg/H9XCZSReMj](https://discord.gg/H9XCZSReMj)
Дані будуть видалені протягом **3 днів**

{% hint style="danger" %}
Видалення даних є незворотним і не підлягає відновленню.
{% endhint %}
