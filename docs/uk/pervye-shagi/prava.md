# Права бота

FistashkinBot потребує певних прав для коректної роботи.

{% hint style="warning" %}
Без потрібних прав частина команд буде недоступна або викличе помилку.
{% endhint %}

{% hint style="info" %}
Ви можете дозволити використання певних команд лише в потрібних каналах або для певних користувачів.
{% endhint %}

**1.** Для цього слід перейти в налаштування сервера з **комп'ютерної** версії Discord до розділу **Інтеграція.**
**2.** Наступним кроком потрібно знайти **FistashkinBot** у списках ботів і натиснути **Керування**, щоб перейти в режим редагування.
**3.** Після цього відкриється вікно з **усіма командами**. Ви можете редагувати різні умови для використання команд тощо.


## Обов'язкові права

| Право | Опис |
|---|---|
| Перегляд каналів | Бот повинен бачити канали |
| Надсилання повідомлень | Відповіді на команди |
| Вбудовувати посилання | Embed-повідомлення |
| Прикріплювати файли | Зображення та файли |
| Читати історію повідомлень | Деякі команди |
| Додавати реакції | Інтерактивні меню |

## Права для модерації

| Право | Команда |
|---|---|
| Кікати учасників | `/kick` |
| Банити учасників | `/ban` |
| М'ютити учасників | `/mute` |
| Видавати попередження | `/warn` (3 з 3 варнів = бан) |
| Керувати повідомленнями | `/purge` |
| Керувати ролями | `/role` |
| Керувати каналами | `/delay` |
