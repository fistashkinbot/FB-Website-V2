# FistashkinBot Changelog 0.4 до 1.8

## Версія 1.8 (частина вже є в боті)
#### Нові команди:
 - `вгадай` — Команда для вгадування чогось по картинці (ігри, міста, логотипи, країни та транспорт)
 - `налаштування автомод` — Команда для керування правилами автомоду бота
 - `моніторинг` — Список моніторингів, на яких є бот для просування
 - `плей` — Відтворює музику в голосовому каналі
 - `стоп` — Зупиняє плеєр
 - `гучність` — Встановлює гучність плеєра
 - `рольінфо` — Відображає інформацію про роль (колір, позицію тощо)
 - `емоція` — Відображає випадковий емодзі з серверів, на яких присутній бот
 - `реакція` — Тепер ви можете плакати, червоніти, танцювати тощо
#### Інше:
 - Команда `рп` тепер називається `взаємодія`
 - Команда `слот` змінена, тепер перед ставкою потрібно натискати кнопку підтвердження
 - До команди `стат` додано додаткову інформацію
 - До системи логування додано дію `Відвідування голосового каналу`
 - Логи тепер надсилаються через вебхуки, а також створюються за їх відсутності
 - Додано перевірки на взаємодії з кнопками
 - Функцію зміни теми бота під Хелловін і Новий рік було переписано
 - Переписано безліч функцій та подій
 - **[Скасовано]** Додано враппер [**FiftyAPI**](https://docs.fifty.su/) для гри `вгадай`
 - **[API]** Додано JSON файли із закритого [**FiftyAPI**](https://docs.fifty.su/) для гри `вгадай`
 - Виправлено рядки у файлах локалізації
 - Запити aiohttp переміщено до директорії services як окремі модулі
 - До команди `юзер` додано кнопку `Відкрити Spotify` якщо учасник слухає музику
 - Команда `взаємодія` зазнала змін
 - **[Веб]** Переписано футер сайту, злегка змінено верхню панель
 - **[Веб]** Змінено домен сайту, тепер він [**fistashkinbot.xyz**](https://fistashkinbot.xyz), старий домен недоступний
 - **[Музика]** Переписано до робочого стану
 - **[Музика]** Замість YouTube треки беруться із SoundCloud
 - **[Музика]** Повністю переписано плеєр
 - **[Музика]** Тепер при пошуку вам буде доступний вибір треку
 - **[Музика]** Lavalink оновлено до версії 4.0
 - Виправлено команди `кейси`, `бійцівський_клуб` та `монетка`
 - **[Бот]** Додано Jishaku 2.6.5
 - У команді `розбан` тепер відображатимуться лише ті учасники, які присутні у списку банів
 - Переписано обробник для команд модерації
 - Переписано модуль `/moderation.py` для зменшення розміру файлу

## Версія 1.7
##### Нові команди:
 - `нсфв` — Команда для перегляду NSFW аніме-контенту
 - `налаштування логи` — Команда налаштування каналу для логування
 - `кубик` — Кубик, на якому випадає випадкове число
 - `аніме-тян` — Перегляд картинок з аніме-тянками
##### Інше:
 - Команди взаємодії перейшли до єдиної — `рп`
 - Відображення ланки (Shard) в інформації про сервер
 - Система логування на сервері. Підтримувані події:
   - Новий учасник приєднався
   - Учасник покинув сервер
   - Нікнейм учасника було змінено
   - Учасник отримав попередження
   - Учасник був забанений
   - Учасник був вигнаний
   - Повідомлення було видалено
   - Повідомлення було відредаговано
 - **[Веб]** Виправлення та зміни на сайті:
   - Оновлено інформацію **🛠️ Мої можливості**
   - Змінено колір сайту (Ребрендинг)
   - Оновлено відображення ембеду сайту в Discord
   - Оновлено бібліотеку FontAwesome
   - При натисканні "Додати до Discord" вкладка відкривається у новому вікні браузера
 - Змінено основний колір бота (Ребрендинг)
   - Змінено колір емодзі в команді `сервер`
 - **[Бот]** Покращення ранкінгу:
   - Виправлено публікацію про підвищення в особисті повідомлення
   - Для досвіду змінено обчислення математичних виразів
 - **[Модерація]** За попередження, таймаут, кік та бан користувач отримує відповідні сповіщення в особистих повідомленнях
 - Виправлено `юзер` не виводив інформацію про ботів
 - У команді `про себе` встановлено ліміт у 2048 символів
 - Команди `видати_роль`/`зняти_роль`/`створити_роль` були прибрані
