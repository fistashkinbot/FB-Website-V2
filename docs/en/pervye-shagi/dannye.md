# Stored Data

FistashkinBot stores only the data necessary for its functionality.

{% hint style="info" %}
We do not sell or share your data with third parties. We store certain data for the bot to function correctly.
{% endhint %}

## What is stored?

| Data | Purpose |
|---|---|
| Discord User ID | Bot Functionality / Moderation / Economy |
| Server ID | Binding bot settings to the server / **AutoMod** |
| Channel IDs | Temporary voice channels / Audit Log |
| Role IDs | Role shop / Audit log |
| Message IDs | Audit Log ( Message was **edited**/**deleted** ) |
| Discord API command data | Request processing |

## What is NOT stored?

- Message content
- Personal information (Email, passwords, Cookie files)
- Voice session history

## Data Deletion

To delete all server data, contact support: [discord.gg/H9XCZSReMj](https://discord.gg/H9XCZSReMj)
Data will be deleted within **3 days**

{% hint style="danger" %}
Data deletion is irreversible and cannot be undone.
{% endhint %}
