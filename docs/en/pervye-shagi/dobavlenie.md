# Adding the Bot

This guide will help you quickly add FistashkinBot to your server.

{% hint style="warning" %}
Adding the bot requires the **Manage Server** permission on the target server.
{% endhint %}

## Steps

1. Go to the [invite link](https://fistashkinbot.xyz/invite)
2. Select your server from the dropdown list
3. Click **Authorize** and complete the captcha
4. Done! The bot will appear on the server 🎉

## After Adding

Make sure the bot has the required permissions. Get started with the command:

```
/help
```

{% hint style="success" %}
All set! FistashkinBot is now on your server. Head to the next sections for configuration.
{% endhint %}
