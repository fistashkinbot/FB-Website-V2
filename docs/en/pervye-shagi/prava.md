# Bot Permissions

FistashkinBot requires certain permissions to function correctly.

{% hint style="warning" %}
Without the required permissions, some commands will be unavailable or will cause an error.
{% endhint %}

{% hint style="info" %}
You can restrict certain commands to specific channels or specific users.
{% endhint %}

**1.** To do this, go to the server settings from the **desktop** version of Discord and navigate to the **Integrations** section.
**2.** Next, find **FistashkinBot** in the bot list and click **Manage** to enter the editing mode.
**3.** A window with **all commands** will open. You can edit various conditions for command usage and more.


## Required Permissions

| Permission | Description |
|---|---|
| View Channels | The bot must be able to see channels |
| Send Messages | Responding to commands |
| Embed Links | Embed messages |
| Attach Files | Images and files |
| Read Message History | Some commands |
| Add Reactions | Interactive menus |

## Moderation Permissions

| Permission | Command |
|---|---|
| Kick Members | `/kick` |
| Ban Members | `/ban` |
| Mute Members | `/mute` |
| Issue Warnings | `/warn` (3 of 3 warns = ban) |
| Manage Messages | `/purge` |
| Manage Roles | `/role` |
| Manage Channels | `/delay` |
