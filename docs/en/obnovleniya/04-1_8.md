# FistashkinBot Changelog 0.4 to 1.8

## Version 1.8 (some features are already in the bot)
#### New commands:
 - `guess` — A command for guessing something from a picture (games, cities, logos, countries, and transport)
 - `automod setup` — A command for managing the bot's automod rules
 - `monitoring` — A list of monitoring sites where the bot is listed for promotion
 - `play` — Plays music in a voice channel
 - `stop` — Stops the player
 - `volume` — Sets the player volume
 - `roleinfo` — Displays information about a role (color, position, etc.)
 - `emoji` — Displays a random emoji from servers the bot is in
 - `reaction` — Now you can cry, blush, dance, and more
#### Other:
 - The `rp` command is now called `interaction`
 - The `slot` command has changed — you now need to press a confirmation button before placing a bet
 - Additional information added to the `stat` command
 - The `Voice Channel Visit` action was added to the logging system
 - Logs are now sent via webhooks, which are also created if they don't exist
 - Added checks for button interactions
 - The function for changing the bot's theme for Halloween and New Year has been rewritten
 - Numerous functions and events have been rewritten
 - **[Cancelled]** Added a wrapper for [**FiftyAPI**](https://docs.fifty.su/) for the `guess` game
 - **[API]** Added JSON files from the closed [**FiftyAPI**](https://docs.fifty.su/) for the `guess` game
 - Fixed strings in localization files
 - aiohttp requests moved to the services directory as separate modules
 - Added an `Open Spotify` button to the `user` command if the member is listening to music
 - The `interaction` command has been updated
 - **[Web]** Rewrote the site footer, slightly modified the top panel
 - **[Web]** Changed the site domain, it is now [**fistashkinbot.xyz**](https://fistashkinbot.xyz), old domain is unavailable
 - **[Music]** Rewritten to a working state
 - **[Music]** Tracks are now sourced from SoundCloud instead of YouTube
 - **[Music]** Player fully rewritten
 - **[Music]** You can now choose a track when searching
 - **[Music]** Lavalink updated to version 4.0
 - Fixed the `cases`, `fight_club`, and `coin` commands
 - **[Bot]** Added Jishaku 2.6.5
 - The `unban` command now only shows members who are in the ban list
 - Rewritten the handler for moderation commands
 - Rewritten the `/moderation.py` module to reduce file size
