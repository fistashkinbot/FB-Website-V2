## Version 1.8

{% hint style="warning" %}
Some changes from version 1.8 have already been implemented in the bot.
{% endhint %}

### Added / New Commands
- `guess` — Object guessing from images (games, cities, logos, countries, vehicles)
- `automod settings` — Management of the bot's auto-moderation rules
- `monitoring` — List of monitoring services where the bot is present for promotion
- `play` — Play music in a voice channel
- `stop` — Stop the music player
- `volume` — Set the player volume
- `roleinfo` — Display information about a role (color, position, and other parameters)
- `emoji` — Show a random emoji from servers the bot is in
- `reaction` — Emotional reactions (cry, blush, dance, etc.)

### Changed
- Command `rp` renamed to `interaction`
- Command `slot` reworked: confirmation via button is now required before betting
- Additional information added to the `stat` command
- Event **"Voice Channel Visit"** added to the logging system
- Logs are now sent via webhooks (if no webhook exists, one is created automatically)
- Added checks for correct button interactions
- The bot's thematic design function (Halloween / New Year) has been completely rewritten
- Significant part of the code for functions and event handlers has been rewritten
- `aiohttp` requests moved to a separate `services` directory
- Added **"Open Spotify"** button to the `user` command (shown if the member is listening to music)
- Command `interaction` received a number of improvements and enhancements
- The `unban` command now displays only members who are actually on the server's ban list
- The moderation commands handler has been completely rewritten
- The `moderation.py` module has been optimized and reduced in size

### API and Integrations
- **[Cancelled]** Adding a wrapper for [**FiftyAPI**](https://docs.fifty.su/) for the `guess` game
- **[API]** Added JSON files from the closed version of [**FiftyAPI**](https://docs.fifty.su/) for the `guess` game

### Web Interface
- **[Web]** Completely rewritten site footer
- **[Web]** Slightly updated top navigation bar
- **[Web]** Site domain changed: now uses [**fistashkinbot.xyz**](https://fistashkinbot.xyz) (old domain is no longer available)

### Music
- **[Music]** Player completely rewritten and brought to working condition
- **[Music]** Tracks are now sourced from **SoundCloud** (instead of YouTube)
- **[Music]** When searching for music, you can now select a specific track from the results
- **[Music]** Lavalink updated to version **4.0**

### Fixed
- Fixed localization strings in all language files
- Fixed commands `cases`, `fight_club`, and `coin`
- Added **Jishaku 2.6.5**
